# cpp_program

A C++ project imported from GitHub (https://github.com/sunny00311/cpp_program).

## Project Structure

- `main.cpp` - Main C++ source file
- `Makefile` - Build configuration using g++ with C++17 standard
- `program` - Compiled binary (generated by make)

## Build & Run

```bash
make        # Compile the program
make run    # Compile and run the program
make clean  # Remove compiled binary
```

## Environment

- **Language**: C++ (C++17)
- **Compiler**: g++ (GCC 14.3.0)
- **Build System**: GNU Make
- **Workflow**: `Start application` runs `make run` as a console application

## Notes

- The GitHub repository was empty at import time; a basic Hello World program was scaffolded to set up the environment.
- Update `main.cpp` and the `Makefile` as needed when adding your own C++ source files.
