A modular **C++ CLI application** that helps students manage studies with **AI-powered planning, explanation, and summarization** using the Gemini API.

---

## 🚀 Key Highlights

- Structured **subject-topic management system**
- Real-time **progress tracking engine**
- AI-powered:
  - 📅 Study plan generation  
  - 📖 Topic explanation  
  - 📝 Notes summarization  
- Clean **OOP architecture (C++17)**
- Persistent storage using file system
- Color-coded CLI for usability
